Proyecto: Conversor de Temperaturas

Descripción:
Este proyecto consiste en crear una página web que permita al usuario ingresar una temperatura y 
seleccionar la unidad de medida (Celsius, Kelvin o Fahrenheit). 
La página debe convertir la temperatura ingresada a las otras dos unidades y mostrar los resultados 
en termómetros gráficos.

Estructura del Proyecto:
- index.html: Archivo HTML principal que contiene la estructura de la página web.
- css/styles.css: Archivo CSS que contiene los estilos para la página web.
- js/script.js: Archivo JavaScript que contiene la lógica para convertir las temperaturas y 
                actualizar los termómetros.

Requisitos:
1. Crear un formulario en HTML que permita al usuario ingresar una temperatura y seleccionar la unidad de medida (Celsius, Kelvin o Fahrenheit).
2. Implementar la lógica en JavaScript para convertir la temperatura ingresada a las otras dos unidades.
3. Mostrar las temperaturas convertidas en termómetros gráficos utilizando CSS para el estilo.


Instrucciones:
1. Crear el archivo `index.html` con el formulario de entrada y los contenedores para los termómetros.
2. Crear el archivo `styles.css` para dar estilo a la página y a los termómetros.
3. Crear el archivo `script.js` para manejar la lógica de conversión de temperaturas y actualizar los termómetros.
4. Probar la página web en diferentes navegadores y dispositivos para asegurarse de que funcione correctamente.
